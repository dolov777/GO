#### 1. Что такое линейный алгоритм?
Линейный алгоритм — это алгоритм, в котором действия выполняются последовательно, одно за другим. В линейных алгоритмах каждый шаг выполняется ровно один раз, и нет ни циклов, ни ветвлений. Это самый простой тип алгоритмов, который обычно применяется для решения простых задач.

#### 2. Пример линейного алгоритма
Рассмотрим пример линейного алгоритма для решения задачи: "Найти сумму двух чисел".

**Шаги алгоритма:**
1. Ввести первое число (a).
2. Ввести второе число (b).
3. Вычислить сумму (c = a + b).
4. Вывести результат (c).

#### 3. Программирование линейного алгоритма
Теперь давайте реализуем этот алгоритм на разных языках программирования.

**Пример на Python:**
```python
# Ввод данных
a = float(input("Введите первое число: "))  # Ввод первого числа
b = float(input("Введите второе число: "))   # Ввод второго числа

# Вычисление суммы
c = a + b

# Вывод результата
print("Сумма двух чисел равна:", c)
```

**Пример на Java:**
```java
import java.util.Scanner;

public class SumTwoNumbers {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Ввод данных
        System.out.print("Введите первое число: ");
        double a = scanner.nextDouble();  // Ввод первого числа

        System.out.print("Введите второе число: ");
        double b = scanner.nextDouble();   // Ввод второго числа

        // Вычисление суммы
        double c = a + b;

        // Вывод результата
        System.out.println("Сумма двух чисел равна: " + c);
    }
}
```

**Пример на C++:**
```cpp
#include <iostream>
using namespace std;

int main() {
    double a, b, c;  // Объявление переменных

    // Ввод данных
    cout << "Введите первое число: ";
    cin >> a;  // Ввод первого числа

    cout << "Введите второе число: ";
    cin >> b;  // Ввод второго числа

    // Вычисление суммы
    c = a + b;

    // Вывод результата
    cout << "Сумма двух чисел равна: " << c << endl;

    return 0;
}
```
