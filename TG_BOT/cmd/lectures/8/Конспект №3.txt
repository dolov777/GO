Тема урока: Формальное исполнение алгоритма

Формальное исполнение алгоритма — это пошаговое выполнение алгоритма вручную, без использования компьютера, с записью промежуточных результатов. Этот метод помогает понять, как работает алгоритм, и отследить изменения значений переменных на каждом шаге.

Как выполнить формальное исполнение алгоритма:

1. Подготовка:
    * Выпишите алгоритм на бумаге.
    * Создайте таблицу для записи значений переменных на каждом шаге. В первой строке таблицы укажите имена переменных, используемых в алгоритме.

2. Исполнение:
    * Начинайте с первого шага алгоритма.
    * Внимательно выполняйте каждую инструкцию алгоритма.
    * После выполнения каждой инструкции записывайте новые значения переменных в следующую строку таблицы. Если значение переменной не изменилось, просто перепишите предыдущее значение.
    * Если в алгоритме есть ветвления (условия), определяйте, какое условие выполняется, и переходите к соответствующей ветви.
    * Если в алгоритме есть циклы, повторяйте шаги внутри цикла, пока условие цикла истинно. Каждый проход цикла записывайте в отдельную строку таблицы.

3. Завершение:
    * Продолжайте выполнять шаги алгоритма до тех пор, пока не достигнете конца алгоритма.
    * Последняя строка таблицы будет содержать окончательные значения переменных.


Пример:

Рассмотрим простой алгоритм вычисления суммы чисел от 1 до 5:

алг сумма_чисел
нач
  цел s, i
  s := 0
  i := 1
  пока i <= 5 выполнять
    s := s + i
    i := i + 1
  все
  вывод s
кон


Таблица формального исполнения:

| Шаг | s | i | Условие i <= 5 |
|---|---|---|---|
| Начало | - | - | - |
| s := 0 | 0 | - | - |
| i := 1 | 0 | 1 | - |
| i <= 5? | 0 | 1 | да |
| s := s + i | 1 | 1 | - |
| i := i + 1 | 1 | 2 | - |
| i <= 5? | 1 | 2 | да |
| s := s + i | 3 | 2 | - |
| i := i + 1 | 3 | 3 | - |
| i <= 5? | 3 | 3 | да |
| s := s + i | 6 | 3 | - |
| i := i + 1 | 6 | 4 | - |
| i <= 5? | 6 | 4 | да |
| s := s + i | 10 | 4 | - |
| i := i + 1 | 10 | 5 | - |
| i <= 5? | 10 | 5 | да |
| s := s + i | 15 | 5 | - |
| i := i + 1 | 15 | 6 | - |
| i <= 5? | 15 | 6 | нет |
| вывод s | 15 | 6 | - |


Результат: В результате формального исполнения мы получили значение s = 15, что является суммой чисел от 1 до 5.

Формальное исполнение алгоритма — полезный инструмент для понимания работы алгоритма и отладки, особенно на начальных этапах изучения программирования. Он помогает развить навыки пошагового анализа и отслеживания изменений значений переменных, что пригодится при написании и отладке собственных программ.
