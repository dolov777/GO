Тема урока: Алгоритмическая конструкция «ветвление»: полная и неполная формы

Ветвление — одна из основных алгоритмических конструкций, позволяющая программе выполнять разные действия в зависимости от условия. 
1. Неполная форма ветвления:

* Описание: В неполной форме ветвления действие выполняется только тогда, когда условие истинно. Если условие ложно, то никакие действия не выполняются, и программа переходит к следующей инструкции после ветвления.
* Блок-схема:

     ┌───────┐
     │ Условие │
     └───┬────┘
         │ да
         ▼
     ┌───────┐
     │ Действие│
     └───────┘


* Пример (псевдокод):

если температура > 25 тогда
  включить кондиционер
конец если


* Пример (Python):

temperatura = 30
if temperatura > 25:
  print("Включить кондиционер")



2. Полная форма ветвления:

* Описание: В полной форме ветвления предусмотрены действия как для истинного, так и для ложного условия. Если условие истинно, выполняется один набор действий, если ложно — другой.
* Блок-схема:

     ┌───────┐
     │ Условие │
     └───┬─┴───┘
         │ да   нет
         ▼     ▼
     ┌───────┐ ┌───────┐
     │ Действие 1│ │ Действие 2│
     └───────┘ └───────┘


* Пример (псевдокод):

если температура > 25 тогда
  включить кондиционер
иначе
  выключить кондиционер
конец если


* Пример (Python):

temperatura = 20
if temperatura > 25:
  print("Включить кондиционер")
else:
  print("Выключить кондиционер")



Ключевые слова:

В разных языках программирования используются разные ключевые слова для реализации ветвления, но общий смысл сохраняется. Например:

* Python: if, else, elif (для нескольких условий)
* Pascal: if, then, else
* C++/Java/C#: if, else


Вложенные ветвления:

Внутри блока одного ветвления может находиться другое ветвление. Это называется вложенным ветвлением и позволяет создавать более сложную логику.

Пример вложенного ветвления (Python):

temperatura = 30
if temperatura > 25:
  print("Жарко")
  if temperatura > 35:
    print("Очень жарко!")
else:
  print("Прохладно")



Ветвление — мощный инструмент для создания программ, которые могут реагировать на разные ситуации и принимать решения на основе данных. Понимание полной и неполной форм ветвления — основа для написания любых программ.
